Project

Infinity Network

Project Description

Infinity Network is a vibrant platform I built for buying, selling, and reviewing used cars. With roles for admins, agents, sellers, and buyers, everyone has a unique experience. Agents manage car listings, buyers browse, save, and rate cars, sellers list their vehicles, and admins oversee everything.
It's a fun, interactive way to navigate the used car market!

Project Purpose and Goal

Infinity Network was built to simplify the buying and selling process while enhancing the experience. By supporting multiple user roles, the platform helps users connect easily, make informed decisions, and enjoy a secure, personalized shopping experience. Our goal is to streamline transactions with ratings, reviews, and personalized recommendations.

Key Goals :

  • Simplify the Process: Agents list cars, buyers search and rate them, and sellers track listings.
  • Enhance User Experience: A smooth, intuitive platform tailored for all users.

Web Stack and Explanation

We used a powerful mix of technologies to build Infinity Network:

  • Languages & Tech:
    • JavaScript & Node.js for backend functionality
    • HTML & CSS for an interactive, responsive design
    • Java with Spring Boot for backend services and API integrations
    • MongoDB & MySQL for managing data efficiently
  • Tools & Frameworks:
    • IntelliJ IDEA for Java and Spring Boot development
    • Postman for API testing
    • GitHub for version control
    • Tiaga for project management
    • Nginx for load balancing and performance optimization
    • AliCloud for hosting and deployment
