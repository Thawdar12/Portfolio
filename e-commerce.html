Project Purpose and Goal

The goal was simple: create a furniture shopping app that's super easy to use and fun to navigate! By integrating third-party APIs and fine-tuning database performance, we wanted to make sure users could browse, shop, and check out with ease while keeping everything smooth and scalable in the backend.

Web Stack and Explanation

  • Backend: Powered by Odoo for all things e-commerce
  • Database: MySQL to manage products and user data
  • Frontend: Built with Flutter for a beautiful, responsive mobile app
  • Tools: Visual Studio Code, Postman, and GitHub for development and teamwork

Problems and Thought Process

The biggest challenge? I was new to Flutter, but I dove in headfirst and quickly got the hang of it! I spent a lot of time crafting that perfect, intuitive experience for users. I also worked hand-in-hand with the backend team to ensure seamless data flow and smooth API integration. On top of that, I regularly checked in with stakeholders, making sure they were happy with progress and kept aligned with their vision for the app.

What I Learned

I got to play with Flutter and Odoo, becoming way more comfortable with both. I learned the art of creating engaging interfaces that make users want to stick around. I also improved my communication skills by keeping stakeholders involved every step of the way, making sure their feedback shaped the app. Along the way, I picked up valuable insights into database optimization, third-party API integration, and building a system that scales as it grows.
