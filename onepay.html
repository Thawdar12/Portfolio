Project

OnePay

Project Description

OnePay is a secure payment gateway built to handle lots of transactions while keeping your data safe. With features like extra layers of login security, data protection, and fraud detection, it's designed to keep everything running smoothly and safely. We focused on making it fast, reliable, and easy to use, so you can trust it for all your payments!

Project Purpose and Goal

OnePay's mission was straightforward: build a robust payment platform capable of managing large-scale transactions effortlessly. We focused on incorporating advanced security features to protect sensitive data and prevent fraud, delivering a dependable and risk-free solution for users. Additionally, we prioritized creating an adaptable, user-friendly interface that ensures seamless navigation and an intuitive experience for users across different devices.

Web Stack and Explanation

  • Backend: Hosted on AliCloud, powered by Node.js and SpringBoot for scalable service development
  • Database: Managed relational data with MySQL and PostgreSQL
  • Frontend: Created responsive user interfaces using HTML and CSS
  • APIs & Security: Integrated RESTful APIs and secure data gateways for smooth communication; implemented AES-256 encryption, MFA, tokenization, and Fraud Detection for comprehensive security
